Hands over.

You work normally

A local AI watches your screen and builds structured understanding of every workflow. Activity classification filters noise. A vector knowledge base links tasks by meaning across sessions. Style analysis captures how you write.

Agents get the real thing

Procedures include strategy, steps, guardrails, your voice, and decision logic. Compiled to Claude Code skills, OpenClaw SOPs, Codex instructions, and a universal MCP server. One observation, every agent format.

You stay in control

Every procedure starts as a draft. You review strategy, steps, and guardrails before anything reaches an agent. Six readiness gates must pass. The system never auto-promotes.

Everything stays local

Local VLM via Ollama. Screenshots deleted after annotation. Auto-redaction of secrets. Encrypted at rest. No cloud, no accounts, no telemetry. Your data never leaves your machine.

Connect any agent.

One command. Works with Claude Code, OpenClaw, Codex, Cursor, Windsurf, or anything that speaks MCP.

$ agenthandover connect claude-code
Done. 3 procedures available as /slash-commands

$ agenthandover connect mcp
MCP server ready. 5 tools exposed to any agent.

$ agenthandover connect openclaw
Synced. 3 procedures in ~/.openclaw/workspace/
