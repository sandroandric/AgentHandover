- **2026-02-16** — Task 5: Artifact store (compress + encrypt + write pipeline). Implemented `ArtifactStore` in `crates/storage/src/artifact_store.rs` with zstd level-3 compression, XChaCha20-Poly1305 authenticated encryption (24-byte nonces), SHA-256-based artifact IDs, date-hierarchical storage (yyyy/mm/dd), and atomic writes (tmp + fsync + rename). Binary format uses OCAA magic header with version byte, nonce, and original size metadata. `store()` compresses then encrypts; `retrieve()` decrypts then decompresses. `artifact_path()` performs recursive directory search with fallback to today's date. Added 3 integration tests covering round-trip store/retrieve, encryption/compression verification, and date hierarchy validation. All 3 tests pass; full workspace (18 tests) clean. Files changed: `crates/storage/Cargo.toml` (added zstd, chacha20poly1305, sha2, hex deps; regex dev-dep), `crates/storage/src/lib.rs` (added `pub mod artifact_store`). Files added: `crates/storage/src/artifact_store.rs`, `crates/storage/tests/artifact_test.rs`.

Added macOS-only dependencies (`core-graphics 0.24`, `core-foundation 0.10`, `accessibility-sys 0.2`) under `[target.'cfg(target_os = "macos")'.dependencies]`. Added 7 integration tests across 3 test files (idle_test.rs, screenshot_test.rs, accessibility_test.rs). All 44 workspace tests pass.
